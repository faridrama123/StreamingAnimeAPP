# Technime Android TV/Mobile Anime Streaming App
This App scrapes links from various anime streaming websites and displays them in the App

# Libraries
* Google Lean back
* Android.x Appcompact
* Glide
* Okhttp, Volley
* JSOUP
* Gson
* All the sites used in the application
